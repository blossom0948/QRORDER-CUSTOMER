# QRORDER Customer

손님용 QR 테이블 주문 사이트입니다.

## 배포 URL

https://blossom0948.github.io/QRORDER-CUSTOMER/

## 역할

- 테이블 정보 표시
- 메뉴 조회 및 검색
- 추천/인기 배지 표시
- 인기/추천 메뉴 빠른 담기
- 메뉴 상세 하단 시트와 기본 옵션 선택
- 한국어, 영어, 일본어, 중국어 보기 모드
- 고정 문구, 메뉴명, 설명, 옵션, 장바구니, 주문 완료 화면 다국어 전환
- 직접 추가한 메뉴의 내장 음식 사전 기반 즉시 번역
- 큰 글씨 접근성 모드
- QR 토큰 확인 상태와 실시간 동기화 상태 표시
- 모바일 하단 주문 확인 바
- 장바구니와 함께 담기 좋은 메뉴 추천
- 주문 요청사항 입력
- 후불/선불 준비 결제 모드
- 주문, 결제, 환불 안내 시트
- 주문 완료 후 5초 카운트다운 뒤 주문 화면 자동 복귀
- 현재 주문의 접수, 조리중, 완료 진행상황 표시
- 진행 중 주문에서 물, 앞접시, 냅킨 직원 호출

사장님용 관리 화면은 별도 저장소 `QRORDER-ADMIN`에서 배포합니다.
