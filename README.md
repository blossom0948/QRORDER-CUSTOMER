# QRORDER Customer

손님용 QR 테이블오더 주문 사이트입니다.

## 배포 URL

https://blossom0948.github.io/QRORDER-CUSTOMER/

## 주요 기능

- QR 링크의 `store`, `table`, `token`으로 매장과 테이블 연결
- 메뉴 조회, 검색, 장바구니, 주문 접수
- 주문 완료 후 5초 카운트다운 뒤 메뉴 화면 복귀
- 현재 주문 진행상황 확인
- 직원 호출
- 한국어, 영어, 일본어, 중국어 보기 모드
- URL에 Firebase QR 정보가 없으면 localStorage 데모 모드로 동작

손님 화면에는 관리자 기능, 메뉴 관리, 매장 설정 화면을 노출하지 않습니다.
